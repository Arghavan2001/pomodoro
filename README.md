# Pomo Notif

**Pomo Notif** is a Chrome extension designed to help you manage your time and boost productivity using the Pomodoro technique.

## Features
- Start, pause, and reset Pomodoro timers.
- Receive desktop notifications when a session ends.
- Simple and user-friendly interface.
- Customizable work and break durations.

## Installation
1. Clone or download this repository.
2. Open Chrome and go to `chrome://extensions/`.
3. Enable **Developer mode**.
4. Click **Load unpacked** and select this project folder.

## Usage
- Open the extension from the Chrome toolbar.
- Configure your preferred work/break times in the Options page.
- Start the timer and focus on your tasks!
